    • logstash记录

    1.拉取镜像
    docker pull logstash

    2.配置文件
    input {
    kafka {
    bootstrap_servers => ‘10.35.22.82:32786’
    group_id => ‘testgroup’
    topics => [‘flumeTopic’]
    }
    }
    output {
    stdout {}
    elasticsearch {
    hosts => “10.35.22.82:9200”
    index => “mprobe_index”
    }
    }

    3.安装插件
    在Dockerfile 配置

    FROM logstash:5.6.4

RUN logstash-plugin install logstash-input-kafka
RUN logstash-plugin install logstash-output-elasticsearch


CMD ["-f", "/some/config-dir/logstash.conf"]

    在构建镜像时执行logstash bin目录的插件安装脚本

    1. 构建镜像：
      docker build -t my-logstash .

    2. 运行容器并挂载配置文件
      docker run -d –name=mylogstash -v /home/aoineko/docker/logstash/logstash.conf:/some/config-dir/logstash.conf:ro my-logstash

    • Spring Cloud <一>

    简介

    Spring Cloud 提供了一系列分布式系统经常使用的模式（例如： 配置管理 ， 服务发现， 断路器， 智能路由， 微代理 ， 控制总线）。首页上的介绍常用模式一直在增加，现在又加入了一次性凭证(one-time tokens)、全局锁(global locks)、领导者选举(leadership election)、分布式sessions、集群状态(cluster状态)。
    使用Spring Cloud开发者可以使用分布式系统开发中催生的各个模板模式，建立自己的服务或应用

    Features

    Spring cloud 致力于提供开箱即用的典型用例和可扩展性机制覆盖
    Distributed/versioned configuration
    Service registration and discovery
    Routing
    Service-to-service calls
    Load balancing
    Circuit Breakers
    Global locks
    Leadership election and cluster state
    Distributed messaging

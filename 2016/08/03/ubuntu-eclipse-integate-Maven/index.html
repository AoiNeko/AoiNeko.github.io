    • Location:http://download.eclipse.org/mpc/indigo/ 安装了Eclipse MarketPlace，从里面搜索Maven Integration 安装一直报错

      Cannot complete the install because one or more required items could not be found.
Software being installed: m2e - Maven Integration for Eclipse 1.4.0.20130601-0317 (org.eclipse.m2e.feature.feature.group 1.4.0.20130601-0317)
Missing requirement: Maven Integration for Eclipse JDT 1.4.0.20130601-0317 (org.eclipse.m2e.jdt 1.4.0.20130601-0317) requires 'bundle org.slf4j.api 1.6.2' but it could not be found

    From: m2e - Maven Integration for Eclipse 1.4.0.20130601-0317 (org.eclipse.m2e.feature.feature.group 1.4.0.20130601-0317)

    推测是org.slf4j.api缺少， 在 http://www.filewatcher.com/m/org.slf4j.api_1.6.4.v20120130-2120.jar.35067-0.html 下载了slf4j.api的jar包 放到eclipse的plugins文件夹下依然报上面的错误

    后来在网上找到eclipse indigo版本，Maven版本大于1.3都会报错。
    在http://download.eclipse.org/technology/m2e/releases/1.3 下载以后终于正常运行

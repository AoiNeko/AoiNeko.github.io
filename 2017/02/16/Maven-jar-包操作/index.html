    • Maven jar 包操作

    添加jar包到本地maven库

    mvn install:install-file 
-DgroupId=FCKeditor 
-DartifactId=FCKeditor 
-Dversion=2.3 
-Dpackaging=jar 
-Dfile=d:\FCKeditor-2.3.jar

    发布jar包到远程mvn库

    mvn deploy:deploy-file
-DgroupId=lotus.domino
-DartifactId=NCSO
-Dversion=7.0
-Dpackaging=jar
-Dfile=E:\dominosso\dominoapp\dominoapp\WEB-INF\lib\NCSO.jar
-Durl=http://yourlocalrepository:8888/archiva/repository/internal
-DrepositoryId=internal

    迁移本地mvn库路径

    maven仓库默认路径是 ${user.home}/.m2/repository, 但是我的笔记本系统盘空间实在不够了
    只能迁移到挂载的windows硬盘上去.

    mv ~/.m2 ~/windisk/

    移动后在maven的setting.xml设置localRepository路径即可

    然后在idea里面settings设置好本地mvn库路径和setting文件位置即可
